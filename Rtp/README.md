# Random Teleport
!!!
Znajdziesz na spwanie znajdź NPC z globusem na głowie i tagiem `RTP` który przeteleportuje was na randomowe miejsce na mapie.
!!!

!!!warning 
Komenda `/rtp` nie istnieje, aby sie przeleportować musisz wrócić na spawn!
!!!
