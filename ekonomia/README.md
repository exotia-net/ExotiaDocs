# Ekonomia

Cała ekonomia jest oparta na systemie GUI, które można uruchomić za 
pomocą komendy `/bank` oraz NPC, którego znajdziesz w urzędzie miasta!

![](https://i.imgur.com/TM7Y2Vj.png)
